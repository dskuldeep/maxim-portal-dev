# Custom Components Examples

This page demonstrates various custom components you can use in your documentation.

import { FAQItem } from '../../components/FAQItem'
import { FeatureCard } from '../../components/FeatureCard'
import { CodeExample } from '../../components/CodeExample'
import { AlertBox } from '../../components/AlertBox'
import { StepCard } from '../../components/StepCard'
import { CardGrid } from '../../components/CardGrid'

## FAQItem Component

The FAQItem component creates an accordion-style FAQ item that can be expanded/collapsed.

<FAQItem 
  question="What is Maxim AI?"
  answer="Maxim AI is an enterprise-grade artificial intelligence platform that empowers developers and businesses to build intelligent applications quickly and efficiently."
/>

<FAQItem 
  question="How do I get started?"
  answer="You can get started by creating an account, obtaining your API key, and installing our SDK. Check out our Quick Start guide for detailed instructions."
  defaultOpen={true}
/>

<FAQItem 
  question="What programming languages are supported?"
  answer={
    <ul style={{ margin: 0, paddingLeft: '1.5rem' }}>
      <li>Python</li>
      <li>Node.js/JavaScript</li>
      <li>Go</li>
      <li>Java</li>
      <li>Ruby</li>
    </ul>
  }
/>

## CodeExample Component

Display code snippets with a copy button.

<CodeExample
  language="bash"
  title="Installation"
  code={`npm install @maxim-ai/sdk
# or
pip3 install maxim-ai`}
/>

<CodeExample
  language="python"
  title="Basic Usage"
  code={`from maxim_ai import MaximAI

client = MaximAI(api_key="your_api_key")
response = client.chat.complete(
    model="maxim-1",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response)`}
/>

## AlertBox Component

Display different types of alerts and notices.

<AlertBox type="info" title="Information">
  This is an informational message. Use this for general tips and helpful information.
</AlertBox>

<AlertBox type="success" title="Success!">
  Your API key has been successfully generated and saved.
</AlertBox>

<AlertBox type="warning" title="Warning">
  Make sure to keep your API keys secure and never commit them to version control.
</AlertBox>

<AlertBox type="error" title="Error">
  Invalid API key. Please check your credentials and try again.
</AlertBox>

## StepCard Component

Create step-by-step instructions with visual numbering.

<StepCard
  step={1}
  title="Create an Account"
  description="Visit our signup page and create your account using your email address or social login."
/>

<StepCard
  step={2}
  title="Get Your API Key"
  description="Once logged in, navigate to your dashboard and generate your first API key from the API Keys section."
/>

<StepCard
  step={3}
  title="Install the SDK"
  description="Choose your preferred programming language and install our SDK using the package manager."
/>

<StepCard
  step={4}
  title="Make Your First Request"
  description="Use the SDK to make your first API call and start building amazing applications!"
  isLast={true}
/>

## Combining Components

You can combine multiple components for rich documentation:

<AlertBox type="info">
  <strong>Pro Tip:</strong> Combine components to create engaging documentation pages!
</AlertBox>

<StepCard
  step={1}
  title="Use FAQItem for Common Questions"
  description="FAQItem components are perfect for answering frequently asked questions in an organized way."
/>

<StepCard
  step={2}
  title="Add Code Examples"
  description="Use CodeExample to show code snippets with copy functionality for better developer experience."
/>

<StepCard
  step={3}
  title="Highlight Features"
  description="FeatureCard components help showcase key features in an attractive grid layout."
  isLast={true}
/>
