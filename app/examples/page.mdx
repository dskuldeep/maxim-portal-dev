# Examples Overview

This section demonstrates various features and components you can use in your documentation.

## Available Examples

### Meta File Examples
- Custom titles and ordering
- Hidden pages
- External links
- Collapsible sections
- Nested structures

### Custom Components
Check out the [Custom Components](/examples/custom-components) page to see:
- **FAQItem** - Accordion-style FAQ items
- **FeatureCard** - Feature showcase cards
- **CodeExample** - Code snippets with copy button
- **AlertBox** - Info, success, warning, and error alerts
- **StepCard** - Step-by-step instructions
- **CardGrid** - Grid of clickable cards

All components are reusable and can be imported in any MDX file!
