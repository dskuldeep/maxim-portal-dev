# Quick Start Guide

Get up and running with Maxim AI in minutes.

## Prerequisites

Before you begin, ensure you have:
- A Maxim AI account ([Sign up here](https://maxim-ai.com/signup))
- Your API key from the dashboard
- Your preferred development environment set up

## Installation

Choose your programming language:

### Python

```bash
pip3 install maxim-ai
```

### Node.js

```bash
npm install @maxim-ai/sdk
# or
yarn add @maxim-ai/sdk
```

### Go

```bash
go get github.com/maxim-ai/go-sdk
```

## Authentication

Store your API key securely using environment variables:

```bash
export MAXIM_AI_API_KEY="your_api_key_here"
```

## Your First Request

### Python Example

```python
from maxim_ai import MaximAI
import os

# Initialize the client
client = MaximAI(api_key=os.getenv("MAXIM_AI_API_KEY"))

# Make a chat completion request
response = client.chat.complete(
    model="maxim-1",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is artificial intelligence?"}
    ],
    temperature=0.7,
    max_tokens=150
)

print(response.choices[0].message.content)
```

### Node.js Example

```javascript
import { MaximAI } from '@maxim-ai/sdk';

const client = new MaximAI({
  apiKey: process.env.MAXIM_AI_API_KEY
});

async function main() {
  const response = await client.chat.complete({
    model: 'maxim-1',
    messages: [
      { role: 'system', content: 'You are a helpful assistant.' },
      { role: 'user', content: 'What is artificial intelligence?' }
    ],
    temperature: 0.7,
    maxTokens: 150
  });

  console.log(response.choices[0].message.content);
}

main();
```

### cURL Example

```bash
curl https://api.maxim-ai.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $MAXIM_AI_API_KEY" \
  -d '{
    "model": "maxim-1",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "What is artificial intelligence?"}
    ],
    "temperature": 0.7,
    "max_tokens": 150
  }'
```

## Understanding the Response

The API returns a JSON response with the following structure:

```json
{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1677858242,
  "model": "maxim-1",
  "choices": [{
    "message": {
      "role": "assistant",
      "content": "Artificial intelligence (AI) refers to..."
    },
    "finish_reason": "stop",
    "index": 0
  }],
  "usage": {
    "prompt_tokens": 28,
    "completion_tokens": 150,
    "total_tokens": 178
  }
}
```
