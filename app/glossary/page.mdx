# Glossary

Definitions of key terms and concepts used throughout Maxim AI documentation.

import { FAQItem } from '../components/FAQItem'

## A

<FAQItem 
  question="API Key"
  answer="A unique identifier used to authenticate API requests to Maxim AI. Keep your API keys secure and never share them publicly."
/>

<FAQItem 
  question="Authentication"
  answer="The process of verifying your identity when making API requests, typically done using an API key in the request headers."
/>

## B

<FAQItem 
  question="Batch Request"
  answer="A method of sending multiple API requests together to improve efficiency and reduce latency."
/>

## C

<FAQItem 
  question="Context Window"
  answer="The maximum number of tokens (words/characters) that can be processed in a single API request. Maxim AI supports context windows up to 128K tokens."
/>

## D
<FAQItem
    question="Data"
    answer="Answer."
/>

## M

<FAQItem 
  question="Model"
  answer="A machine learning model that processes input and generates output. Maxim AI offers various models optimized for different use cases."
/>

## T

<FAQItem 
  question="Token"
  answer="A unit of text that the AI model processes. Approximately 1 token = 4 characters or 0.75 words in English. Token usage determines API costs."
/>

<FAQItem 
  question="Temperature"
  answer="A parameter (0-1) that controls the randomness of AI responses. Lower values (0.1-0.5) produce more consistent outputs, while higher values (0.7-1.0) produce more creative outputs."
/>
